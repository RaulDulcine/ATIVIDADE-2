# ATIVIDADE-2
segunda entrega de atividade no github SENACPE2026.1

## **Objetivo**

Ler códigos com erros e fragilidades, identificar problemas, corrigir e justificar melhorias.

Em todos os desafios, você deve responder:

1. Qual era o erro?
2. Como você corrigiu?
3. Por que sua solução é melhor?

4. # **Orientação para a entrega**

Em cada desafio, você deve entregar em um documento (Word, Google Drive, OneDrive, etc.):

## **1. Qual era o erro?**

Descrever o problema encontrado.

## **2. Como você corrigiu?**

Mostrar a correção feita.

## **3. Por que sua solução é melhor?**

Explicar por que o novo código está mais correto, claro ou seguro.
